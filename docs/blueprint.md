# **App Name**: EduVerse

## Core Features:

- Digital Books: Provide access to digital books for classes 3-10.
- Solutions: Provide step-by-step solutions for textbook exercises.
- Recorded Classes: Offer access to recorded classes.
- Notes: Allow students to create and save notes within the app.
- Doubt Section: Provide a dedicated section for students to ask and get their doubts cleared. This is a bulletin-board feature only; it does not provide real-time assistance.
- Help Contact: Offer a help/contact section for student inquiries and support.
- AI Paper Generator: AI-based tool that generates practice papers (MCQ + Long Answer Tests) based on selected subjects and topics.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey trust and intelligence.
- Background color: Light gray (#F0F0F0) for a clean and distraction-free interface.
- Accent color: A bright green (#90EE90) to highlight important elements and CTAs.
- Font pairing: 'Poppins' (sans-serif) for headlines and short text, 'PT Sans' (sans-serif) for body text, creating a balance of modernity and readability.
- Use clear and intuitive icons relevant to educational topics; icons should be consistent throughout the app.
- Design a clean, mobile-first layout that is easy to navigate and optimized for smaller screens, considering the Indian student user base.
- Incorporate subtle animations and transitions to enhance user experience and provide feedback during interactions.
